1. Необходимые пакеты:
	pip3 install google-api-python-client 
	pip3 install oauth2client
	pip install psycopg2

2. Ограничения:
	Таблица сканируется только в диапазоне A1:D100. Если диапазон выходит за границы, то требуется внести корретировку в скрипт
	Адрес на сайте ЦБ остается неизменным, равно как и формат выдачи данных
	В целях обеспечения целостности БД, данные в БД записываются только после того, как все поля в таблице будут заполнены. При удалении хотя бы одного значения данные из БД стираются. Первичный ключ - номер заказа. Заказы с одинаком номером существовать не могут, в БД записывается последний экземпляр заказа.
